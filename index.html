HTML is called a "markup language" as opposed to a "programming language", because it doesn't allow to program the computer and make it do everything you want, but only to mark text up with tags in order to structure it. The markup you write within the text tells the web browser (for instance, Google Chrome, Mozilla Firefox, Internet Explorer, Safari, …) how to display the text to the user in the webpage.
